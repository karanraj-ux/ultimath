# 🐱 Catbot - Interactive AI Companion Platform

## Product Vision Statement

**Catbot transforms AI chat from a boring Q&A tool into an engaging, social, and gamified experience where users interact with cute, animated AI companions in group debates, complete story-driven missions, and build meaningful connections through collaborative fact-checking and shared adventures.**

### 3. AI Fact-Checking System

#### How It Works
1. **User makes a claim** in chat
2. **Other users can challenge** with "Fact Check" button
3. **AI analyzes** the claim using multiple sources
4. **AI provides verdict**:
   - ✅ True (with sources)
   - ❌ False (with corrections)
   - ⚠️ Partially True (with context)
   - ❓ Unverifiable (needs more info)
5. **Verdict is final** - Cannot be edited by users
6. **Sources shown** - Transparent and traceable
7. **Can be shared** - Export as image or link

### AI Fact-Checking System

#### Architecture

```
User Request
    ↓
Edge Function: fact-check
    ↓
1. Extract claim from message
    ↓
2. Query multiple sources:
   • Wikipedia API
   • Wolfram Alpha
   • Google Search API
   • Academic databases
    ↓
3. AI Analysis (GPT-4/Gemini):
   • Compare sources
   • Identify contradictions
   • Assess reliability
   • Generate verdict
    ↓
4. Format result:
   • Verdict (True/False/Partial)
   • Explanation
   • Sources with links
    ↓
5. Store in database
    ↓
6. Return to client
    ↓
Display in chat (immutable)
```

#### Edge Function: fact-check

```typescript
// supabase/functions/fact-check/index.ts
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts';

interface FactCheckRequest {
  claim: string;
  roomId: string;
  messageId: string;
}

serve(async (req) => {
  const { claim, roomId, messageId } = await req.json();

  // 1. Query multiple sources
  const sources = await Promise.all([
    queryWikipedia(claim),
    queryWolframAlpha(claim),
    queryGoogleSearch(claim),
  ]);

  // 2. AI analysis
  const analysis = await analyzeWithAI(claim, sources);

  // 3. Generate verdict
  const verdict = {
    claim,
    verdict: analysis.verdict, // 'true', 'false', 'partial', 'unverifiable'
    explanation: analysis.explanation,
    sources: analysis.sources,
    confidence: analysis.confidence,
  };

  // 4. Store in database
  await supabase.from('fact_checks').insert({
    room_id: roomId,
    message_id: messageId,
    ...verdict,
  });

  // 5. Return result
  return new Response(JSON.stringify(verdict), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

```typescript
      schema: 'public',
      table: 'room_messages',
      filter: `room_id=eq.${roomId}`,
    },
    (payload) => {
      // New message received
      addMessageToChat(payload.new);
      
      // Trigger companion animation
      if (payload.new.sender_type === 'ai_persona') {
        animateCompanion('talking');
      }
    }
  )
  .subscribe();

// Presence system (who's online)
const presenceChannel = supabase
  .channel(`presence:${roomId}`)
  .on('presence', { event: 'sync' }, () => {
    const state = presenceChannel.presenceState();
    updateOnlineUsers(state);
  })
  .subscribe();
```
