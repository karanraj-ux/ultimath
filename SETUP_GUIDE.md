# AI Chat Platform - Setup Guide

## 🚀 Quick Start (3 Steps)

### Step 1: Add Your API Keys
1. Click "API Keys" in the navigation
2. Add keys for the AI providers you want to use:
   - **OpenAI** (GPT-4, GPT-4o, GPT-3.5) - Get from https://platform.openai.com/api-keys
   - **Google Gemini** (Gemini 2.5, 1.5) - Get from https://makersuite.google.com/app/apikey
   - **Groq** (Llama, Mixtral) - Get from https://console.groq.com/keys
   - **Anthropic Claude** (Claude 3.5, 3) - Get from https://console.anthropic.com/

### Step 2: Start a Conversation
1. Go to Home page
2. Click "New Chat"
3. Start typing or upload an image!

### Step 3: Share Your Conversations
1. Click the Share button in any conversation
2. Copy the link
3. Share on WhatsApp, email, or anywhere!

## 🆘 Troubleshooting

### "Please add your API key first"
- Go to API Keys page
- Add a key for the provider you're using
- Make sure the key is valid

### "Failed to get AI response"
- Check your API key is correct
- Make sure you have credits with the AI provider
- Try a different model

### Image upload not working
- Make sure image is under 5MB
- Use JPG, PNG, or WebP format
- Switch to a vision-capable model

### Can't see shared conversation
- Check the link is correct
- Try refreshing the page
- Link might have been deleted
